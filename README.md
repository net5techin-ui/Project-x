# TN28 Men's Fashion Store - 🚀 Amazon Style Backend

**Your premium menswear store is now connected to a unified Cloud/Local backend architecture.**

### 🌐 Storefront (View Website)
[Visit Your Store](http://localhost:3000)
- View new arrivals, featured collections, and trending styles.
- Professional search and checkout experience.
- Mobile-responsive design.

### 🏠 Admin Control Panel
[Access Admin Panel](http://localhost:3000/admin.html)
- **Manage Products**: Add, Edit, or Delete inventory in real-time.
- **Media Library**: WordPress-style image management (Drag & Drop).
- **Dashboard**: Track total revenue, brands, and top-selling items.
- **Order Management**: Connect to Cloud to view all customer orders.

### 🛠️ Developer Commands
- Run locally: `npm run dev`
- Build for deployment: `npm run build`
